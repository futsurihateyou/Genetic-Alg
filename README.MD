# README: Оптимизационные алгоритмы (Генетический алгоритм и Метод роя частиц)

## Введение
Программа реализует генетический алгоритм (ГА) и метод роя частиц (МРЧ, PSO) для тестирования на четырех функциях: Sphere, Rosenbrock, Rastrigin, Ackley. Интерфейс на JavaFX позволяет настраивать параметры и проводить эксперименты. Результаты экспериментов сохраняются в JSON.

Требования: JDK 21+ (Adoptium Temurin), IntelliJ IDEA. Зависимости: org.json, OpenJFX.

## Использование интерфейса
Интерфейс состоит из трех вкладок. Вводите параметры и нажимайте кнопки для запуска.

### 1. Вкладка "Генетический алгоритм"
Запускает ГА для минимизации функции.

- Размер популяции: Количество решений (пример: 100).
- Скорость мутации: Вероятность изменений (пример: 0.01).
- Поколения: Число итераций (пример: 100).
- Функция: Выбор из списка.
- Размерность: Число переменных (пример: 2).

Кнопка "Запустить ГА": Выводит позицию и значение функции.

Пример: Популяция 50, мутация 0.02, поколения 200, функция Ackley, размерность 3.

### 2. Вкладка "Метод роя частиц"
Запускает МРЧ для оптимизации.

- Размер роя: Количество частиц (пример: 50).
- C1/C2: Коэффициенты (пример: 2.0).
- Итерации: Число шагов (пример: 100).
- Использовать LBEST: Галочка для локального поиска.
- Функция: Выбор.
- Размерность: Пример: 2.

Кнопка "Запустить МРЧ": Выводит позицию и значение.

Пример: Рой 30, C1/C2 2.0, итерации 150, LBEST включен, функция Sphere, размерность 4.

### 3. Вкладка "Эксперименты"
Проводит тесты ГА и МРЧ на всех функциях (фиксированные параметры).

- Размерность: Общая (пример: 2).
- Запусков на алгоритм/функцию: Число повторений (пример: 10).

Кнопка "Запустить эксперименты": Сохраняет результаты в JSON.

Пример: Размерность 3, запусков 5.

## Почему значение функции в ГА сильно меняется при каждом запуске
ГА — стохастический алгоритм: инициализация популяции и операции (селекция, кроссовер, мутация) зависят от случайных чисел. Разные запуски приводят к разным траекториям поиска, особенно на мультимодальных функциях. Для стабильности увеличьте параметры или усредните результаты по нескольким запускам.
